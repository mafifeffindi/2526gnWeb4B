📝 CSS Comments

Pengertian: CSS Comments digunakan untuk menambahkan catatan/penjelasan di dalam kode CSS. Komentar tidak ditampilkan di browser dan tidak mempengaruhi tampilan halaman. Berguna untuk dokumentasi dan debugging.

Sintaks Dasar

/* Ini adalah single-line comment */

/*
  Ini adalah multi-line comment.
  Bisa ditulis lebih dari satu baris.
*/

body {
  /* warna latar belakang */
  background-color: #f0f4f8;
}

Jenis-jenis Penggunaan Comment

JenisContohKegunaan
Single-line/* komentar */Penjelasan singkat
Multi-line/* baris 1
baris 2 */		Penjelasan panjang
Section Header/* === HEADER === */Pemisah bagian kode
Nonaktifkan kode/* color: red; */Debug sementara

⚠️ CSS Errors

Pengertian: CSS Errors adalah kesalahan penulisan sintaks CSS yang menyebabkan properti tidak berfungsi. Browser biasanya mengabaikan deklarasi yang salah tanpa menampilkan pesan error di halaman.

Contoh Kesalahan Umum

❌ Error 1: Lupa Titik Koma (Missing Semicolon)
❌ Salah
p {
  color: red
  font-size: 16px;
}
✅ Benar
p {
  color: red;
  font-size: 16px;
}
❌ Error 2: Typo Nama Properti
❌ Salah
div {
  bakground-color: blue;
}
✅ Benar
div {
  background-color: blue;
}
❌ Error 3: Lupa Satuan (Missing Unit)
❌ Salah
p { font-size: 16; }
div { margin: 10; }
✅ Benar
p { font-size: 16px; }
div { margin: 10px; }
❌ Error 4: Kurung Kurawal Tidak Lengkap
❌ Salah
h2 {
  color: green;
/* lupa tanda } */
✅ Benar
h2 {
  color: green;
}

Cara Mendeteksi Error

CaraKeterangan
Browser DevTools (F12)Tab "Styles" — properti salah akan dicoret
W3C CSS Validatorjigsaw.w3.org/css-validator
Linting (VS Code)Ekstensi Stylelint

📐 CSS Margins

Pengertian: CSS Margin adalah ruang kosong di luar border elemen. Digunakan untuk memberi jarak antar elemen. Berbeda dengan padding yang berada di dalam border.

Margin Individual

margin-bottom: 25px;   margin-left: 60px;

margin-left: 60px

Shorthand & margin: auto

margin: 5px 20px 30px 50px;         /* atas kanan bawah kiri */
margin: 0 auto;                      /* tengahkan elemen */

SintaksArti

box-sizing: content-box vs border-box

/* content-box: width = lebar konten saja (default) */
/* border-box: width = sudah termasuk padding & border */

🔗 CSS Links

Pengertian: CSS Links mengatur tampilan elemen <a> menggunakan pseudo-class untuk berbagai kondisi/state: link, visited, hover, dan active.

Link States (Pseudo-class)

a:link { color: blue; } /* belum dikunjungi */
a:visited { color: purple; } /* sudah dikunjungi */
a:hover { color: red; } /* mouse di atas */
a:active { color: orange; } /* saat diklik */

  2. Item dua

  2. Item dua
