# 📖 Judul Aplikasi

**Sistem Pendataan Buku**

Aplikasi ini dibuat untuk memudahkan pengelolaan data buku menggunakan PHP dan MySQL. Pengguna dapat menambah, melihat, mengubah, dan menghapus data buku.

---

# ✨ Fitur Aplikasi

- 📋 Menampilkan daftar buku
- ➕ Menambah data buku
- ✏️ Mengedit data buku
- 🗑️ Menghapus data buku
- 💾 Data tersimpan di MySQL
- 🎨 Tampilan berwarna hijau

# ▶️ Cara Menjalankan

1. Jalankan Apache dan MySQL pada XAMPP.
2. Import file `database.sql` ke phpMyAdmin.
3. Simpan folder di:

```
C:\xampp\htdocs\UAS-PWEB
```

4. Buka browser.

```
http://localhost/UAS-PWEB/
```
