# 📖 Deskripsi

Sistem Pendataan Mahasiswa merupakan aplikasi berbasis web yang dibuat menggunakan **HTML5**, **CSS3**, **Bootstrap 5**, **PHP Native**, dan **MySQL**.

Aplikasi ini digunakan untuk mengelola data mahasiswa dengan konsep **CRUD (Create, Read, Update, Delete)** sehingga pengguna dapat menambah, melihat, mengubah, dan menghapus data mahasiswa.

---

- 📋 Daftar Mahasiswa

# 🚀 Cara Menjalankan Project

1. Install XAMPP.
2. Jalankan Apache dan MySQL.
3. Import file **database.sql** ke phpMyAdmin.
4. Simpan folder project di dalam folder **htdocs**.
5. Jalankan browser.
6. Buka alamat berikut.

```

http://localhost/SistemPendataanMahasiswa/

```
