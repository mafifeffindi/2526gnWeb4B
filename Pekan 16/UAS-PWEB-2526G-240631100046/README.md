# Sistem Pendataan Mahasiswa

## Deskripsi
Website Sistem Pendataan Mahasiswa merupakan aplikasi sederhana berbasis PHP dan MySQL yang digunakan untuk mengelola data mahasiswa. Aplikasi ini dibuat untuk memenuhi tugas Ujian Akhir Semester (UAS) mata kuliah Pemrograman Web.

## Fitur

- Home
- Tambah Data Mahasiswa
- Daftar Mahasiswa
- Edit Data Mahasiswa
- Hapus Data Mahasiswa

## Teknologi

- HTML5
- CSS3
- PHP Native
- MySQL
- XAMPP

## Struktur Folder

```
uas-mahasiswa
│
├── config
├── css
├── function
├── proses
├── database.sql
├── index.php
├── tambah.php
├── daftar.php
├── edit.php
├── hapus.php
└── README.md
```

##Cara Menjalankan

1. Jalankan Apache dan MySQL pada XAMPP.
2. Import file `database.sql` ke phpMyAdmin.
3. Simpan project pada folder `htdocs`.
4. Buka browser.
5. Akses:

```
http://localhost/uas-mahasiswa
```
